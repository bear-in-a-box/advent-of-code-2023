# Advent of Code 2023

## Things to do after cloning

```
nvm use # or just have Node.js 20 installed
npm ci
```

## Running

### Current day in watch mode

```
./watch.sh <part>
```

(`<part>` can be either `a` or `b`)

### Manually

```
npx nodemon days/<day>/<part>.ts
```
